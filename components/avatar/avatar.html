Avatar

Avatar is mostly useful when representing profile pictures. They can also be used when a user likes on social media handles. We can also use badges on avatar's to represent user status.

Avatar can also be presented in different ways. Some of them are

  1. Standard Avatar - Normal avatar with class name as .avatar
  2. Avatar with different sizes - Avatars with 5 different sizes with classname as .md,>.sm, .md, .lg, .xl, .xxl
  3. Squared Avatar - Avatar's that are square in shape with class name as .square
  4. Avatar with badges - Avatar's representing status of user, with class name as .dnd, .online, .offline
