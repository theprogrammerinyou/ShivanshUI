Image

There are two types of images present in this component library.

  1. Responsive Image - Use a classname img-resp to make an image responsive
  2. Round Image - Use classname img-round to make an image round

Responsive Image

          
            <img
              src="your link goes here"
              alt="responsive-img"
              class="img-resp"
            />

Round Image

          
            <img
              src="your link goes here"
              alt="round-img"
              class="img-round"
            />
