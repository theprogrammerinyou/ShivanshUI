Input

Input fields are most important when you have forms, or want to take input from the user.

There are 4 types of inputs available in the component library

  1. Primary Input - A normal input you can use input just for normal styling.
  2. Success Input - A input with success message, use a div as well with classname success-div, as well use success along with the normal input class.
  3. Warning Input - A input with success message, use a div as well with classname alert-div, as well use alert along with the normal input class.
  4. Error Input - A input with success message, use a div as well with classname error-div, as well use error along with the normal input class.

Primary Input

          
            <div class="label-inputs">
              <label for="username" class="success-div">
              Password * </label>
              <input
                class="success input"
                id="username"
                type="text"
                placeholder="Success"
              />
              <div class="success-div">Password validation met!
