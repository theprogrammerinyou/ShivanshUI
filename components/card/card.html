Cards

Cards are bordered-divs with some information inside them. You would see lot of cards when you visit an E-commerce website.

This component library has 6 types of cards. Try the examples to check the cards.

  1. Card with basic text - Use classname card to display a basic card. Also use the inner classes card-content for better styling of the content inside the card.
  2. Card with dismiss button on top - Use a image with class cross-img to have the dismiss card feature.
  3. Card with badges - Use a div with classname badge to get a badge over the basic card.
  4. Card with text overlay - Use a image on background to get the text over it.
  5. Card with horizontal text - Use class name horizontal-card along with the basic card classes to get a horizontal card with image beside it.
  6. Card with shadow. - All these cards have some shadow in them, this makes the cards look better.
