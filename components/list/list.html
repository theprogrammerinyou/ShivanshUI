Lists

Lists can be used to represent anything. They can be used in notification lists, they can be used to show some number of things on screen and many more things.

In this component library, we have

  • Spaced List - Use classname flex to align list spaced between each other.
  • Stacked List - Use classname stacked-list on ul to get stacked list as well as use classname stacked-list-item on li to get desired notification stacks.

Spaced List

  • List Item 1
  • List Item 2
  • List Item 3
        
          <ul class="flex">
            <li>List Item 1</li>
            <li>List Item 2</li>
            <li>List Item 3</li>
          </ul>
