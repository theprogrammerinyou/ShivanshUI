Button

Buttons are of many types. Some of them are present here

  1. Normal Button - Use classname button, btn-primary
  2. Link Button - Provides a link to a page, a href should be given compulsorily. button, btn-link
  3. Icon Button - These buttons have a icon beside it that represent the action they do. button, btn-icon
  4. Fab Button - These buttons are easily representable by the icons that they have, they do not need a text with it.button, btn-fab 
          
            <button class="button btn-primary">Primary Button
            <button class="button btn-link">
              <a href="link goes here">
                Link Button </a>
            </button>
            <button class="button btn-icon">
               <img src="link goes here" alt="delete-icon" 
               width="20" height="20" />Icon Button
            </button>
