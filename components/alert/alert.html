Alert

Alert's are messages which give some special message. They can be easily understood by the color they have.

There are 4 different alerts in the component library.

  1. Info Alert - .alert-primary class should be used
  2. Error Alert - .alert-error class should be used
  3. Success Alert - .alert-success class should be used
  4. Warning Alert - .alert-warning class should be used
