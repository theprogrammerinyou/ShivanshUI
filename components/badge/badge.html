Badge

Badges are usually numbers, represented on top of profile images, or icons, or on top of anything, that show a number. Usually they are used to count number of notifications user has received.

This component library has 2 types of badges.

  1. Profile with badge - can be used with div with classname as profile-badge. A class name as img should also be given to the image tag.
  2. Icon with badge - can be used with div with classname as icon-badge. This can also be combined with .sm .md, .xl, .xxl, .lg to increase the size of the icons
